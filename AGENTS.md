# AGENTS

- This is a Vite + Three.js browser game.
- Three.js must be installed from npm.
- Do not use copyrighted NFL, team, player, logo, or uniform names.
- Use procedural low-poly models and generated browser audio.
- Local play only for v0.1.0.
- Do not add Netlify or Render.
- Keep systems simple and arcade-style.
- Preserve `npm run install:all`, `npm run play`, and the root scripts.
- Run `npm run check` and `npm run build:client` before final commit.
- No backend server is needed.
- Keep the tone chunky, retro, and original.
